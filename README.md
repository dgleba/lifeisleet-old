## Development

### Requirements

  * Ruby 2.2.2
  * PostgreSQL 9.4

### Setup

    $ git clone https://github.com/lifeisleet/lifeisleet.git && cd $_
    $ bin/setup
    $ bin/rails s

### Testing

    $ bin/rspec
